# MIDORI-DOCS

Official website and documentation platform for the MIDORI programming language.

## Stack

- Next.js (App Router)
- TypeScript
- Vercel deployment target

## Local Development

```bash
npm install
npm run dev
```

## Quality Checks

```bash
npm run typecheck
npm run build
```

## Content Layout

- `content/docs/*.md`: main documentation pages
- `content/blog/*.md`: blog entries
- `content/versions/*`: versioned docs snapshots

## Brand Assets

- `public/brand/logo/midori-logo.png`: primary logo
- `public/brand/og/`: OpenGraph images

## Deployment

Vercel is the intended deploy target for production and preview deployments.
